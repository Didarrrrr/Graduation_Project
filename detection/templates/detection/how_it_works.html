How It Works

Understanding the science behind our image forgery detection system

Detection Process Overview

Our system uses four complementary digital image forensics techniques to detect forgeries. No machine learning is involved — just proven forensic analysis methods that have been validated by the digital forensics community.

Copy-Move Detection (Block-Based DCT)

Copy-move forgery duplicates content from one area of an image to another area in the same image. Our method divides the image into overlapping blocks, extracts DCT-based low-frequency signatures, and clusters consistent displacement vectors to identify repeated regions.

What We Detect
  • Duplicated texture/object fragments within a single image
  • Consistent geometric displacement between matched keypoint groups
  • Suspicious regions with confidence based on cluster support
Best Use Case

Strong for in-image duplication and object hiding. A low score can still happen for splicing or global edits, where no duplicated region exists in the same image.

Noise Analysis (PRNU-Inspired)

This method estimates residual sensor-like noise by subtracting a denoised version from the original image. It then checks local noise statistics across patches and flags regions with inconsistent noise signatures.

What We Detect
  • Patch-level residual noise anomalies
  • Local inconsistencies that may indicate inserted or manipulated regions
  • Suspicious area ratio and region confidence scores
Noise analysis is especially useful when metadata is missing or stripped.

Error Level Analysis (ELA)

What is ELA?

Error Level Analysis (ELA) is a forensic technique introduced by Neal Krawetz in 2007. It works by identifying areas in an image that have different compression levels, which can indicate manipulation.

How Does It Work?

The uploaded image is resaved at a fixed JPEG compression level (95%). This creates a new version of the image with consistent compression.

The original image is compared pixel by pixel with the resaved version. The difference between them represents the "error level."

Regions where the error level differs significantly from surrounding areas are flagged as potentially tampered. These appear as brighter areas in the ELA image.
Key Insight

When an image is edited and resaved, the edited regions will have different compression characteristics than the original regions. ELA makes these differences visible.

Limitations
  • Works best on JPEG images
  • May produce false positives on high-quality originals
  • Cannot detect all types of manipulation
  • Should be used with other analysis methods

Metadata Analysis

What is EXIF Metadata?

EXIF (Exchangeable Image File Format) metadata is embedded information in image files that records details about the camera, settings, and processing history of the image.

What We Look For
Editing Software
Signatures of Photoshop, GIMP, etc.
Timestamp Inconsistencies
Mismatched creation and modification times
Camera Information
Missing or inconsistent camera data
Common Suspicious Indicators
High Risk Adobe Photoshop, GIMP detected in software field
Medium Risk Modification date differs from creation date
Medium Risk Missing camera make/model information
Low Risk Social media processing detected
Note: Metadata can be stripped or modified. Its absence doesn't necessarily mean an image is forged, but its presence of editing signatures is a strong indicator of manipulation.

Interpreting Results

Score Fusion: Overall confidence is computed as ELA ~40% + Metadata ~20% + Copy-Move ~20% + Noise ~20%. Trust adjustments (sparse EXIF, few copy-move matches, noisy texture) and compositing-style reweighting may apply; see the fusion line on each result.
Authentic
Confidence Score: 0-39%

The image shows no significant signs of manipulation across ELA, metadata, copy-move, and noise analysis indicate the image is likely original.

Suspicious
Confidence Score: 40-69%

Some anomalies detected. The image may have minor edits or the analysis is inconclusive. Further review of per-method scores is recommended.

Forged
Confidence Score: 70-100%

Strong indicators of manipulation detected. Multiple suspicious regions found by one or more forensic methods.

Important Disclaimer: This system provides an automated analysis based on digital image processing techniques. While these methods are scientifically validated, no automated system is 100% accurate. For critical applications such as legal proceedings, always consult with a certified digital forensics expert.

Types of Image Forgery We Detect

Copy-Move Forgery

Parts of the image are copied and pasted elsewhere in the same image to hide or duplicate objects. This is primarily detected by block-based DCT copy-move matching and supported by ELA/noise inconsistencies.

Image Splicing

Parts from different images are combined to create a composite image. This often leaves traces in error levels and metadata inconsistencies.

Image Retouching

Minor manipulations like removing blemishes, adding/removing objects, or adjusting colors. These edits often have different compression characteristics than the original.
