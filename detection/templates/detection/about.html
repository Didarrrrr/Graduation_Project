Project Overview
Problem Statement

The rapid growth of social networking platforms and the widespread availability of powerful image editing tools such as Adobe Photoshop have made image manipulation easier than ever before. Forged images pose serious threats — they can spread misinformation in the media, be used as fabricated evidence in legal proceedings causing injustices, and fuel cyberbullying on social platforms.

There is a clear and urgent need for a system that can detect digital image forgery reliably and make this capability accessible to non-experts such as journalists, students, and everyday social media users.

Our Solution

This Image Forgery Detection System provides a user-friendly web application that allows users to upload images and receive a detailed analysis report indicating whether the image has been forged, along with a visual depiction of the forged regions.

The system uses a hybrid multi-algorithm forensic pipeline:

  • Error Level Analysis (ELA): Detects compression inconsistencies
  • Metadata Analysis: Examines EXIF data for editing traces and integrity
  • Copy-Move Detection (Block-Based DCT): Detects duplicated regions inside the same image
  • Noise Analysis (PRNU-inspired): Detects local noise inconsistencies from tampering

Final confidence is computed by weighted fusion: ELA ~40%, Metadata ~20%, Copy-Move ~20%, and Noise ~20%.
